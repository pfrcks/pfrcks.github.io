Distance Metrics and Machine Learning

2018/11/28

Getting Started

A distance function(also referred as a metric) in mathematics refers to the function that defines a distance between any two points of a set. The most prevalent example is the Euclidean distance. But let’s first define what makes a function a distance function.

For a function to be a distance function, it has to satisfy the following properties:

  • Non-negativity : $d(x, y) \geq 0$
  • Identity : $d(x, y) = 0 \iff x=y$
  • Symmetry : $d(x, y) = d(y, x)$
  • Triangle Inequality : $d(x, z) \leq d(x, y) + d(y, z)$

With that out of the way, lets take a look at some of the common distances

$L_p$ distances

For two vectors $a, b$ in $\mathbb{R}^d$, the $L_p$ distances are defined as $d_p(a, b) = ||a-b||_p = \left(\sum_{i=1}^d\left(|a_i - b_i|\right)^p\right)^\frac{1}{p}$

  • L=2 : This is the most common case. For L = 2, this transforms to Euclidean distance, which is used extensively both in Machine Learning and in sciences: $d_2(a, b) = ||a-b||_2 = \left(\sum_{i=1}^d\left(|a_i - b_i|\right)^2\right)^\frac{1}{2}$

  • L=1 : This is referred to as Manhattan Distance, called so because to walk in Manhattan you cannot cut through streets: $d_1(a, b) = ||a-b||_1 = \sum_{i=1}^d\left(|a_i - b_i|\right)$

  • L = 0 : Referred to as Hamming Distance which essentially checks for equality between the points: $d_0(a, b) = ||a-b||_0 = \sum_{i=i}^d\mathbb{1}\left(a=b\right)$

  • L = inf : Returns only the largest magnitude in the vector: $d_\infty(a, b) = ||a-b||_\infty = \max_{i=1}^d|a_i - b_i|$

Jaccard Distance

Jaccard distance measures how similar two sets are. The Jaccard distance for two sets A and B is defined as

$$d_J(A, B) = 1 - JS(A, B)=1 - \frac{|A \cap B|}{|A \cup B|}$$

Cosine Distance

Measures the cosine of the angle between two d-dimensional vectors in $\mathbb{R}^d$ $$d_{cos}(A, B) = 1 - \frac{\sum_{i=1}^d a_ib_i}{||a|| ||b||}$$

Edit Distance

Edit distance is the number of delete or insert operations needed to convert a string A to a string B.
